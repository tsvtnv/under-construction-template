## ✨ What is this?

A clean, minimal under construction page template. Zero dependencies — just drop `index.html` into your project or hosting provider and you're live in seconds. Perfect for holding a domain while your site is being built.

## 🔥 Features

- **⚡ Zero dependencies** — Pure HTML + CSS, works everywhere
- **📱 Responsive** — Looks great on all screen sizes
- **🎨 Easy to customise** — Edit the HTML directly, no build step needed
- **🚀 Deploy anywhere** — Works on GitHub Pages, Netlify, Vercel, any static host

## 🚀 Getting Started

```bash
git clone https://github.com/tsvtnv/under-construction-template.git
```

Open `index.html` in your browser — or drag it straight to your hosting provider. That's it.

**Customise:**
- Edit the text in `index.html`
- Change colours in `style.css`
- Swap the illustration for your own
